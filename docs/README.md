## Contexto del sistema

La aplicación permite crear pedidos personalizados de bandeja paisa desde una página web local. El usuario puede seleccionar ingredientes, definir cantidades por porción o gramos, elegir una bebida, gestionar un carrito temporal, enviar el pedido y consultar posteriormente los pedidos creados con su estado.
